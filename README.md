# Nader - لعبة الأفعى

لعبة أفعى (Snake) بسيطة بواجهة عربية تعمل مباشرة في المتصفح.

## هل يمكن تشغيلها هنا؟
نعم، يمكن تشغيلها محليًا داخل هذا المشروع ثم فتحها في المتصفح.

## طريقة التجربة بسرعة

### الخيار 1 (الأفضل): سكربت جاهز للتشغيل
```bash
./run_game.sh
```
أو مع منفذ مخصص:
```bash
./run_game.sh 9000
```

### الخيار 2: تشغيل خادم محلي يدويًا
```bash
python3 -m http.server 8000
```
ثم افتح الرابط:
- `http://localhost:8000`

### الخيار 3: فتح الملف مباشرة
يمكنك فتح `index.html` مباشرة في المتصفح (لكن الخيار الأول/الثاني أفضل لضمان سلوك متسق).

## التحكم داخل اللعبة
- الأسهم: تحريك الأفعى
- WASD: بديل للأسهم
- زر **ابدأ / إعادة تشغيل**: بدء اللعبة من جديد

## الملفات
- `index.html`: هيكل الصفحة والواجهة العربية.
- `styles.css`: التنسيق والشكل العام.
- `script.js`: منطق اللعبة (الحركة، التصادم، النقاط، أفضل نتيجة).
- `run_game.sh`: تشغيل سريع للخادم المحلي.
